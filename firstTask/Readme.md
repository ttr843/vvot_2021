## Getting Started


#### 1. Set up credentials file

Win: username/.aws

Unix: $HOME

#### 2. Put albums.json into  bucket


#### 3. Run this command

```bash
./mvn clean clean compile package  
```

#### 4. Run jar with command

```bash
nohup *path to jdk*/java -Xmx32G -jar *jar name*.jar &  
```
